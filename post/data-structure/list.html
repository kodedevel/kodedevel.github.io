بررسی list

از list در برنامه نویسی برای نگهداری عناصری که نوعشون یکسانه استفاده میکنیم.

از list به عنوان آرایه ی دینامیکی نیز یاد میشه، یعنی یک آرایه با طول متغیر و قابلیت حذف عناصر ازش و اضافه کردن عناصر بهش.

توجه

list در زبان های مختلف با نام های مختلفی پیاده سازی شده مثلا در جاوا و کاتلین به عنوان ArrayList پیاده سازی شده و در api قدیمی جاوا به نام vector پیاده سازی شده است؛ همچنین در بعضی از زبان ها (مثل javascript و php) طول آرایه ثابت نیست و از آرایه به عنوان list استفاده میشه.

ساختار list

در تصویر زیر n تا عنصر داخل list نگهداری شده:

به تعداد عناصری که list داخل خودش نگهداری میکنه، size (اندازه) list میگیم.

اندازه ی list برابر با تعداد عناصریه که نگهداری میکنه در تصویر اندازه ی list برابر با n است.

ساختار لیست مشابه ساختار آرایه است هر عنصر داخل یک خونه نگهداری میشه.

به شماره خونه ی هر عنصر index میگیم.

شماره خونه ی اولین عنصر 0 و شماره خونه ی اخرین عنصر n-1 است.

با افزودن یک عنصر به list اندازه ی list یک واحد افزایش پیدا میکنه و با حذف یک عنصر، size (اندازه) list یک واحد کاهش پیدا میکنه.

اگه تعداد عناصری که میخوایم استفاده کنیم ثابت باشه، استفاده از آرایه بهینه تره اما اگه تعداد عناصر متغیر باشه یعنی قصد حذف یا اضافه کردن عناصر رو داشته باشیم استفاده از list بهینه تره.

افزودن عناصر به list

عناصر در List مانند آرایه ها نگهداری میشن با این تفاوت، هر عنصری که اضافه می کنیم، اندازه ی List افزایش پیدا میکنه.

به طور عمومی در زبان های برنامه نویسی با تابعی مشابه add(e: E) یک عنصر رو به List اضافه می کنیم.

اگه بخوایم عناصر رو به لیست اضافه کنیم با افزودن هر عنصر، یک واحد به اندازه ی List اضافه میشه:

  • اولین عنصری که اضافه میکنیم، داخل خونه ی شماره 0 نگهداری میشه و اندازه ی List میشه 1.
  • دومین عنصری که اضافه می کنیم، داخل خونه ی شماره 1 نگهداری میشه و اندازه ی List میشه 2.
  • سومین عنصری که اضافه می کنیم داخل خونه ی شماره 2 نگهداری میشه و اندازه ی List میشه 3.

و به همین ترتیب اگه n عنصر به یک List خالی اضافه کنیم، n امین عنصر داخل خونه ی n-1 نگهداری میشه و اندازه ی list میشه n.

فرض کنید یک list داریم که عناصر e0 و e1 رو نگهداری میکنه و میخوایم دو عنصر e2 و e3 رو بهش اضافه کنیم، با افزودن هر عنصر یک واحد به اندازه ی list اضافه میشه؛ در زیر به صورت تصویری بیان شده.

توجه

به طور عمومی هنگام ایجاد یک List، اندازه ی اولیه ای براش وجود داره به عنوان مثال در زبون جاوا اندازه ی اولیه ی List برابر 5 است و این اندازه هنگامی که تعداد عناصر از 5 بیشتر میشه به طور خودکار افزایش پیدا میکنه.

مثال

میخوایم یک List تعریف کنیم و عناصری از نوع String رو بهش اضافه کنیم.

ArrayList<Integer> list = new ArrayList();
list.add("Lee");
list.add("Rose");
list.add("Joe");
list.add("Billy");
list.add("Susan");
list.add("Emilly");
list.add("Nick");

System.out.println("Size of list after adding elements is " + list.size());
        
val list: ArrayList<Int> = ArrayList();
list.add("Lee")
list.add("Rose")
list.add("Joe")
list.add("Billy")
list.add("Susan")
list.add("Emilly")
list.add("Nick")

println("Size of list after adding elements is ${list.size}")

صدا زدن عناصر از list

به طور عمومی در اکثر زبونا برای صدا زدن عناصر داخل list از تابعی مشابه get(int: index) استفاده میکنیم، که index شماره ی خونه ی عنصر مورد نظر است.

فرض کنید در مثال قبل میخوایم Joe و Susan رو صدا بزنیم.

String name = list.get(2);
String name1 = list.get(4);
        
val name = list.get(2)
val name1 = list.get(4)

میتونیم تمام عناصر رو با استفاده از حلقه ها صدا بزنیم.

for(int i=0; i<list.size(); i++){
  String name = list.get(i);
  System.out.print(name + " ");
}
        
for(i in 0 until list.size ){
  val name = list.get(i)
  print("$name ");
}

حذف عناصر از list

با حذف هر عنصر از list، اندازه ی list یک واحد کاهش پیدا می کنه؛ هنگامی که تمامی عناصر از list حذف بشن اندازه ی list صفر میشه و اصطلاحا میگیم list خالی شده است.

به طور کلی در زبان های برنامه نویسی با استفاده از تابعی به نام remove(index: Int) می تونیم یک عنصر رو از لیست حذف کنیم و بجای index شماره ی خونه ی عنصر مورد نظر رو می نویسیم.

هنگامی که یک عنصر رو از list حذف میکنیم، از یک واحد از شماره خونه (index) عناصر بعدی و یک واحد از اندازه ی list کم میشه.

فرض کنید یک list داریم که n تا عنصر (از e0 تا e(n)) رو داخل خودش نگهداری کرده ابتدا میخوایم عنصر e1 رو حذف کنیم و سپس عنصر e2.

وقتی عنصر e1 رو حذف میکنیم از index عناصر e2 تا e(n) یک واحد کم میشه و اندازه ی list میشه n-1.

بعد از حذف e1، هنگامی که میخوایم عنصر e2 رو حذف کنیم، از index عناصر e3 به بعد یک واحد کم شده و از اندازه ی list میشه n-2

در زیر به صورت تصویری شرح داده شده است.

در مثال قسمت قبل فرض کنید میخوایم Lee, Rose و Nick رو حذف کنیم.

list.remove(0);
list.remove(0);
list.remove(4);
        
list.remove(0)
list.remove(0)
list.remove(4)

خلاصه

  • از list برای نگهداری عناصری با نوع یکسان استفاده میکنیم.
  • میتونیم هر عنصر داخل list رو که خواستیم به صورت تصادفی صدا بزنیم.
  • بر خلاف آرایه ها طول list ثابت نیست و بعد از ایجاد یک لیست میتونیم عناصر رو بهش اضافه کنیم یا ازش حذف کنیم.
