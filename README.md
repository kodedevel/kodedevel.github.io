### نقشه ی سایت

ابتدا سایت با jekyll کامپایل میشه و سپس صفحات کامپایل شده به سه دسته ی کلی تقسیم میشن:

1- post: صفحات سایت در اینجا قرار دارن

2- resources: منابع مورد نیاز کلاینت برای رندر صفحات

3- index.html: صفحه ی اول سایت
